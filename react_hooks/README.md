#运行说明

1, 下载 node

https://nodejs.org/zh-cn/download/

2, 下载 yarn

https://classic.yarnpkg.com/en/docs/install#windows-stable

3, 安装依赖

yarn install --registry https://registry.npm.taobao.org/
    

4, 启动项目

yarn start
